# /figma — Implement from Figma Design

Convert Figma designs into production-ready React components.

## Usage

```
/figma [Figma URL or design description]
```

## Process

1. **Analyze** design — decompose into components, extract tokens
2. **Plan** component hierarchy and implementation order
3. **Implement** atoms → molecules → organisms → pages
4. **Style** with Tailwind CSS (pixel-perfect fidelity)
5. **Test** each component with unit tests
6. **E2E** test the complete user flow
7. **Review** accessibility and responsiveness
